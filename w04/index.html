    Chapter 8

    You can use document.forms to access the forms on a page. Access a specific form use the bracket notation.

    You can also use the form.elements notation to access the elements of a form, or you can get a specific element by using its given name attribute.

    You can use event listeners to alter and control what is shown in a forms input field.

    Chapter 12

    Parentheses are not always required when initializing a new instance of a class. It is only needed if you are passing values to the class creation.

    A constructor function is used to create an instance of a class.

    The 'this' keyword is used to access attributes and methods defined in the class.

    A static method can be called. It is different because it is called by the class instead of an instance of the class.

    Using the 'extends' keyword you can define a second class that inherits all of the attributes and methods from previous class.
